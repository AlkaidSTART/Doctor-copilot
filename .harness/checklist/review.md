# 代码审查检查清单

## 代码质量
- [ ] 代码符合编码规范
- [ ] 变量/函数命名清晰
- [ ] 注释适当且有价值
- [ ] 代码结构清晰
- [ ] 避免重复代码

## 功能正确性
- [ ] 实现符合需求
- [ ] 边界条件处理
- [ ] 错误处理完善
- [ ] 数据一致性保障
- [ ] 并发安全考虑

## 安全
- [ ] 无 SQL 注入风险
- [ ] 无 XSS 风险
- [ ] 无 CSRF 风险
- [ ] 敏感数据保护
- [ ] 权限控制正确

## 性能
- [ ] 避免不必要的计算
- [ ] 数据库查询优化
- [ ] 缓存策略合理
- [ ] 资源使用合理
- [ ] 懒加载/按需加载

## 测试
- [ ] 测试覆盖率达标
- [ ] 测试用例合理
- [ ] 测试命名规范
- [ ] 集成测试覆盖

## 可维护性
- [ ] 模块职责清晰
- [ ] 依赖关系合理
- [ ] 代码易于理解
- [ ] 文档已更新
