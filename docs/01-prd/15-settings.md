# PRD-15 设置

## 背景
设置页承载用户级偏好与基础配置。

## 为什么
统一设置入口可减少运维与支持成本。

## 目标
支持个人偏好、通知偏好、基础安全设置。

## 非目标
- 不提供组织级系统设置（见 Admin）。

## 范围
用户 Profile、通知偏好、会话管理。

## 流程图（Mermaid）
```mermaid
flowchart LR
  A[加载当前配置] --> B[编辑]
  B --> C[校验]
  C --> D[保存并生效]
```
