# PRD-14 消息中心

## 背景
消息中心聚合通知、系统公告与流程回执。

## 为什么
分散消息入口会增加遗漏概率。

## 目标
提供统一收件箱、筛选、批量已读与跳转。

## 非目标
- 不做即时聊天系统。

## 范围
站内收件箱与消息状态管理。

## 流程图（Mermaid）
```mermaid
flowchart TD
  A[消息入库] --> B[用户筛选]
  B --> C[阅读/处理]
  C --> D[状态回写]
```

## ASCII 图
```text
Inbox -> Filter -> Open -> Action -> Mark
```

## 表格
| 消息类型 | 默认保留期 |
|---|---|
| 告警 | 180 天 |
| 任务通知 | 90 天 |
| 系统公告 | 365 天 |
